Avatar Effects : styling users' avatars never was so easy.

== Description ==

Avatar Effects allows styling users' avatars as you want.
The plugin has many settings which allow to control avatar effects, styling and customization.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/avatars-effects` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Avatar Effects screen to configure the plugin

== Frequently Asked Questions ==

= Will this plugin works with profile plugins? =

Yes! The plugin will work with all profile plugins if that plugin coded with wordpress standards.
Meaning that the profile plugin you are going to use calling wordpress "get_avatar" hook.

= Styling not take effect on avatars =

This will happen only if the theme/or plugin does not call wordpress get avatar function mentioning above.

Wordpress get avatar function/hook : <https://codex.wordpress.org/Function_Reference/get_avatar>
